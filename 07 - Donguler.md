##DÖNGÜLER

### 01 – For Döngüsü
- Döngü tanımlanırken, birden fazla değişken veya şart ifadesi kullanılabilir. 

```cs
for (int i = 0, j = 0; i < 10 && j < 5; i++, j++)
{
    Console.WriteLine(i + " " + j);
}
```
